Earwig's Copyvio Detector Language Settings

This tool attempts to detect copyright violations in Wikipedia articles. In search mode, it will check for similar content elsewhere on the web using Google, external links present in the page, or Turnitin (via EranBot), depending on the selected options. In comparison mode, the tool will compare the article to a specific webpage without making any searches, like the Duplication Detector.

Be aware that other websites can copy from Wikipedia, so check the results carefully, especially for older or well-developed articles. Specific websites can be skipped by being added to the excluded URL list.
